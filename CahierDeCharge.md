Design Process
Follow this workflow strictly:
1.	Analyze the three reference websites.
2.	Extract:
o	color palette
o	typography
o	spacing system
o	motion language
o	layout principles
o	visual rhythm.
3.	Define all design tokens before creating any component.
4.	Create wireframes for the full landing page.
5.	Build each section progressively.
6.	Use Nano Banana to generate all required visuals.
7.	Use 21st.dev components whenever possible.
8.	Implement animations using GSAP, Framer Motion, ScrollTrigger, and Lenis.
9.	Capture screenshots of the landing page at 1440px width after completing each section.
10.	Compare every screenshot against the reference websites.
11.	Refine spacing, typography, proportions, imagery, and animations until the overall experience reaches the same—or a higher—level of visual quality.
12.	Do not consider the page complete until it achieves a polished, premium, studio-quality finish.
The final result should feel like an award-winning landing page that could be featured on Awwwards or CSS Design Awards while remaining fast, accessible, responsive, and production-ready.
